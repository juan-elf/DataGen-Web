Platform

An agent that queries your data like a teammate would.

Point DataGen at any SQLite database or PostgreSQL / Supabase connection — no schema config required. On connect it auto-profiles every table and folds row counts, null rates, and semantic types straight into its own system prompt.

Ask a question in plain language and the agent generates SQL, executes it, self-corrects on errors, and can drop into a sandboxed pandas environment for correlation, anomaly detection, and distribution stats — with Tavily web search for context the database doesn't have.

What matters isn't the chat window. It's the trust boundary: every row, web result, and upload is tagged untrusted before it reaches the model, and the write path never lets the LLM emit SQL.
